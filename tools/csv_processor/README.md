# Star Resonance Tool - CSV Processor Kit

このディレクトリは、ゲーム「Star Resonance」のスクリーンショットからデータを抽出し、`master_data.csv` を作成するためのツールセットです。
**AIエージェント（Antigravity, Claude, ChatGPT, Gemini等）にこのディレクトリの `AI_AGENT_MANUAL.md` を読み込ませることで、高精度なCSV作成を依頼できます。**

## ディレクトリ構成

- **`AI_AGENT_MANUAL.md`**: **重要**。AIエージェントへの作業指示書（SOP）。これをAIに渡してください。
- **`input/`**: 解析したいスクリーンショット（.png）をここに置いてください。
- **`assets/master.png`**: アイコン識別のためのマスタ画像（辞書）。
- **`Workflows/csv_creation.md`**: スラッシュコマンド `/csv_creation` の定義ファイル。
- **`Workflows/csv_init.md`**: 環境初期化コマンド `/csv_init` の定義ファイル。
- **`master_data.csv`**: 出力されるCSVファイル（UTF-8, カンマ区切り）。

## 使い方 (User Guide)

### 1. 準備
- `input/` フォルダに、データ化したいスクリーンショットを全て入れます。

### 2. AIへの指示
お使いのインターフェースに応じて、以下のいずれかの方法で実行してください。

#### A. Antigravity（サポーター/開発者モード）を使用する場合
チャット欄で以下のコマンドを入力するだけで、自動的に画像解析とCSV作成が開始されます。
> `/csv_creation`

#### B. 一般的なAIエージェント（ChatGPT/Claude等）を使用する場合
**【重要】AIの精度低下（ハルシネーション）を防ぐため、作業は5〜10枚ずつ分割して依頼してください。**

**初回（1回目のチャット）の指示手順:**
1. 以下のファイルをアップロードしてください（※AIの添付ファイル上限に注意してください。通常は合計10ファイルまでです）。
   - `AI_AGENT_MANUAL.md`
   - `RULES.md`
   - ご自身のPC上にある `master_data.csv` （※初回は本ツール付属の空ファイル、またはヘッダー行のみ作成したもの）
   - ご自身のPC上にある `batch_log.md` （※初回は何も書かれていない空のテキストファイルを作成）
   - `input/` 内の画像のうち **「最初の5〜6枚」**（添付上限を超えない範囲）
2. 以下のプロンプトを入力してください：
   > 「`AI_AGENT_MANUAL.md` の手順に従い、アップロードした画像データを順番に処理してください。**【重要】怠け癖や省略を防ぐため、必ず「画像を1枚開く → 推論ログを書く → CSV行を書く」というループを1枚ずつ完結させてください。** 終わったらCSVとログを出力してください。」

**2回目以降（新しいチャット）の指示手順:**
1. AIが処理を終えて出力した `master_data.csv` と `batch_log.md` の内容（コードブロックのテキスト、またはダウンロード生成物）を、**ご自身のPC上にある同名ファイルにコピペ等で上書き保存**します。（推論ログも必要に応じて保存）
2. **必ず現在開いているチャットを閉じ、新しいチャット（New Chat）を開いてください。**（記憶をリセットするため）
3. 新しいチャットに、以下のファイルを**再度すべて**アップロードしてください。（※batch_log.mdがないとAIがどこまでやったか分からなくなります）
   - `AI_AGENT_MANUAL.md`
   - `RULES.md`
   - **更新したご自身のPC上の `master_data.csv`**
   - **更新したご自身のPC上の `batch_log.md`**
   - **「次の5〜6枚の画像」**（添付上限を超えない範囲）
4. 以下のプロンプトを入力してください：
   > 「`AI_AGENT_MANUAL.md` の手順に従い、データの続きを作成します。アップロードしたCSVとログを読み込み、新しく提供した画像のデータを追記してください。**【重要】必ず「画像を1枚開く → 推論ログを書く → CSV行を書く」というループを1枚ずつ完結させてください。省略は厳禁です。**」
これをすべての画像が終わるまで繰り返してください。

## 注意事項
- **本手法の強み (AI-Driven Visual Extraction)**:
    - 従来のOCRプログラムでは誤認識しやすいゲーム特有のアイコンや数値を、AIが「目」で見て厳密に判断します。
- **環境構築不要**:
    - ユーザーはPython環境やライブラリをインストールする必要はありません。AIとの対話だけで完結します。
