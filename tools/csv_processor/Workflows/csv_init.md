description: ツールの初期化を行います。既存の `master_data.csv`、`batch_log.md`、推論ログをリセットし、最初からやり直す準備をします。
---
1. ユーザー定義の `tools/csv_processor/master_data.csv` ファイルを初期化します。
// turbo
2. 以下の文字列（ヘッダー行＋改行）を `tools/csv_processor/master_data.csv` に書き込んでください。
```csv
モジュールタイプ,ID,魔法耐性,物理耐性,極・HP凝縮,極・絶境守護,極・HP変動,極・HP吸収,筋力強化,敏捷強化,知力強化,特攻ダメージ強化,精鋭打撃,特攻回復強化,マスタリー回復強化,集中・詠唱,集中・攻撃速度,集中・会心,集中・幸運,極・ダメージ増強,極・適応力,極・応急処置,極・幸運会心
```
// turbo
3. `tools/csv_processor/batch_log.md` ファイルの内容を空にして初期化してください。
// turbo
4. `tools/csv_processor/` ディレクトリ直下にある、過去に生成された推論ログファイル（`reasoning_log_*.md` のような名前のファイル等、以前の解析による出力ファイル）を検索し、存在する場合はすべて削除してください。
// turbo
5. 初期化完了をユーザーに報告し、次のアクション（例：`/csv_creation` を実行して新しい解析を開始する）を促してください。
