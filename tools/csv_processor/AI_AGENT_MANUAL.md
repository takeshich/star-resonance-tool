# AI Agent Manual CSV Extraction Workflow (SOP)

## 概要
このドキュメントは、AIエージェントが**視覚機能のみ**を使用してゲーム画面のスクリーンショットからデータを抽出し、CSVファイルを作成するための標準作業手順書（SOP）です。
ユーザー環境への依存を排除するため、**スクリプトの作成・実行は一切行わず、すべてのデータ抽出をAIエージェントの内部処理（Visual Inspection）で行います。**

## 1. 事前準備 (Initial Setup)
作業を開始する前に、以下のファイルが存在することを確認してください。

- **Master Data**: `tools/csv_processor/master_data.csv` (出力先)
    - ヘッダー行が正しいことを確認:
      `モジュールタイプ,ID,魔法耐性,物理耐性,極・HP凝縮,極・絶境守護,極・HP変動,極・HP吸収,筋力強化,敏捷強化,知力強化,特攻ダメージ強化,精鋭打撃,特攻回復強化,マスタリー回復強化,集中・詠唱,集中・攻撃速度,集中・会心,集中・幸運,極・ダメージ増強,極・適応力,極・応急処置,極・幸運会心`
- **Batch Log**: `tools/csv_processor/batch_log.md` (進捗記録用)
- **Reference Image**: `assets/master.png` (アイコン辞書)
    - **重要**: 全てのアイコン識別はこの画像を基準に行います。

## 2. 抽出ルール (Strict Rules)
データの並び順や数値の読み取りについて、以下のルールを**絶対厳守**してください。

1.  **モジュールタイプの判定 (Visual Shape)**
    - アイコンの形状から判定し、CSVの先頭列に記入します。
    - **丸型 (Round)** → `支援`
    - **ひし形 (Diamond)** → `防御`
    - **逆三角形 (Inverted Triangle)** → `攻撃`

2.  **数値の読み取り (Visual OCR)**
    - 各アイコンの横にある数値を読み取ります。
    - **上限値**: 1つの項目の数値が **10** を超えることはありません。
    - **注意**: 「13」や「15」に見える場合は、近接した別の項目（例: 10と5）が合算されている可能性があります。必ず分離してください。

3.  **項目のユニーク制約**
    - 1つの行（モジュール）に同じ項目が2回出現することはあり得ません。
    - 似たアイコンでも必ず別の項目です。マスタ画像と微細な特徴（本/時計の模様など）を照合してください。

## 3. 作業手順 (Execution Steps)

AIエージェントは以下のループを実行してデータを処理します。

### Step 1: 画像の確認 (View)
`view_file` ツールを使用し、`tools/csv_processor/input/` 内の画像を1枚ずつ（またはメモリ許容量内で複数枚）開きます。

### Step 2: データ抽出 (Extract)
開いた画像を目視し、脳内でデータを構造化します。
- `ID`: 連番 (例: Mod_001, Mod_002...)
- `Type`: 形状判定
- `Stats`: アイコンマッチングと数値読み取り

### Step 3: CSVへの書き込み (Write)
抽出したデータをCSVフォーマットのテキスト行に変換し、`master_data.csv` に書き込みます。
- **ツール**: `write_to_file`
- **モード**: 追記 (`Overwrite=False` が使えない場合は `read_file` → 内容結合 → `write_to_file` の手順で上書き更新)

### Step 4: ログ更新 (Log)
`batch_log.md` に処理したファイル範囲とステータスを記録します。
- 例: `| 202X-XX-XX | Batch X (Image_A - Image_B) | ✅ Completed |`

## 4. エラー対応
- **判別不能な画像**: 無理に推測せず、ログに「Skipped (Unclear)」と記録して飛ばしてください。
- **ツールエラー**: `write_to_file` 失敗時は、ファイルの中身を再度確認してからリトライしてください。

---
**このワークフローは、ユーザーの環境構築の手間をゼロにし、AIエージェントの能力のみで完結させることを目的としています。**
